# drones-service

Musalasoft Drones

#### Setup project
Install project using the following command
```bash
.\mvnw clean install
```

#### Run Tests
To run tests, run the following command
```bash
.\mvnw test
```

#### Run Project

```bash
.\mvnw spring-boot:run
```

#### Swagger Documentation
API documentation
```http
  GET /swagger-ui
```
